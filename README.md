# Go Projects

A collection of Go projects built to practice and explore the Go programming language.

## Projects

| Project | Description |
|---------|-------------|
| [github-activity](./github-activity) | CLI tool to fetch and display a GitHub user's recent public activity |
| [task-cli](./task-cli) | Command-line task manager with add, list, update, delete, and status tracking |
| [contact-book](./contact-book) | Command-line contact manager with add, list, search, and delete by ID |

## Getting Started

**Prerequisites:** Go 1.21+

Clone the repository:

```bash
git clone https://github.com/rsmharsha/Go-Projects
cd Go-Projects
```

Navigate to any project directory and build/run it. See each project's README for specific instructions.
